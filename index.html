Information about the parameters

The following parameters can be set to cluster data:

  1. Clustering Algorithm: A list of clustering algorithms available for experiment.
  2. Number of clusters: Expected number of code vectors / clusters. Can be unset when choosing Mean Shift clustering algorithm.
  3. Kernel Radius (R): Radius for the flat kernel of Mean Shift algorithm.
  4. Population Size (S): Size of the solution population of Genetic Algorithm.
  5. Number of iterations (T): Number of Genetic Algorithm iterations (max number of generations).
  6. Sub-sampling (%)*: Set to a floating point value from 0 to 1 to limit the size of the input data set. Example: Data with 1000 vectors and input limit factor is 0.25 will result in only 250 vectors loaded to the memory for clustering.
Note: The "Sub-sampling (%)" field should be set due to limited capability of the web browser when processing large amount of data on client size. In order to overcome problem with large data sets, please try running the scripts using a JavaScript engine such as Node.js for higher performance (index.js is a file for running using Node.js).
